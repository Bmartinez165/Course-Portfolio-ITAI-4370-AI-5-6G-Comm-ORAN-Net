# Reflection: My Growth in Telecommunications

At the start of this course, I understood telecommunications mainly as the technology that makes phone calls, internet access, and communication between devices possible. As the course continued, I learned that telecommunications involves much more, including wireless systems, network architecture, data transmission, cloud technology, 5G Core, and the future development of 6G.

One of the most important things I learned was that 5G is not only about faster internet speeds. It also supports lower latency, more connected devices, and services that need reliable real time communication. Learning about Open RAN showed me how network infrastructure can become more flexible and how different vendors can play a role in building network systems.

This course helped me understand how quickly telecommunications is changing. It also showed me that technical knowledge needs to be paired with research, problem solving, and communication skills. In the future, I want to continue learning about network technologies and explore more ways AI can make networks more efficient and dependable.
