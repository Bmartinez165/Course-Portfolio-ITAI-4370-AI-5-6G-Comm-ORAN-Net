# Reflection: Applying AI to Telecommunications

This course helped me understand that AI can play an important role in telecommunications. Instead of networks only responding after problems occur, AI can help analyze large amounts of network data, recognize patterns, predict potential issues, and support faster decisions.

Through the lab activities and my midterm project, I worked with Python and Jupyter Notebooks. This gave me practice using data and code to explore telecommunications related problems. AI can be used for network traffic prediction, resource optimization, fault detection, security monitoring, and improving the customer experience.

One of my biggest areas of growth was learning how to connect technical concepts to real world use cases. I also improved my ability to work with code, analyze information, and document what I learned. Moving forward, I would like to build stronger Python and machine learning skills and gain more experience working with realistic network datasets.
